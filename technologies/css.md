Kaskadowe arkusze stylów to języki opisujące wygląd elementów w języku znaczników, takim jak HTML. CSS to jedna z trzech technologii, które stanowią podstawę sieci.

W CSS używamy selektorów, aby opisać, na które elementy chcemy wpływać, a następnie deklarujemy styl elementów w bloku.

Na przykład, jeśli chcemy ustawić kolor wszystkich fragmentów tekstu na niebieski, wykonujemy następujące czynności.

```css
p {
    color: blue;
}
```
